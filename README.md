# STEM Study Abroad Checklist

A web-based checklist application designed to help STEM students at Texas State University navigate the study abroad process, with a special focus on the Gilman Scholarship application.

## Purpose

STEM students often face unique challenges when planning to study abroad, including:
- Fitting international experiences into demanding course schedules
- Ensuring technical credits transfer properly
- Managing scholarship applications like the Gilman Scholarship

This app provides a structured, step-by-step guide to help students stay organized throughout the study abroad process.

## Features

- **Interactive Checklist**: Track your progress through the study abroad application process
- **Progress Tracking**: Visual progress bar showing overall completion
- **Gilman Scholarship Section**: Dedicated checklist for Gilman Scholarship applicants
- **Local Storage**: Your progress is automatically saved in your browser
- **Mobile Responsive**: Works on desktop, tablet, and mobile devices
- **Reset Functionality**: Clear all progress and start fresh if needed

## Sections

1. **Research & Planning**: Initial steps for exploring study abroad programs
2. **Application Process**: Tasks for completing study abroad applications
3. **Gilman Scholarship**: Comprehensive checklist for Gilman applicants
4. **Pre-Departure**: Final preparations before leaving

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome icons
- Local Storage API

## How to Use

1. Visit the live site at [https://jeaflynn.github.io/Gilman/]
2. Check off tasks as you complete them
3. Your progress is automatically saved
4. Return anytime to update your progress
5. Use the "Reset All" button to clear progress if needed

## Installation

No installation required! This is a web-based application accessible through any modern web browser.

To run locally:
1. Clone this repository
2. Open `index.html` in your web browser
